[[../../../biosensors|biosensors]] are analytical devices that convert a biological response to a different type of signal that we can measure, via [[../../../transducer|transduction]]. Usually has a biological element as a receptor/recognition site (like [[../../../antibodies|antibodies]]), and a transducer

[[../../../biopuces|biochips]] are physical devices with multiple biosensors, allowing analysis of multiple biological markers at the same time.  useful for high-throughput methods for gene expression profiling or protein detection, often using [[../../../Fluorophores|fluorescence]] for signal detection

[[../../../lab-on-chip|lab-on-chip]] can integrate multiple functions and biochips into a single chip, usually via [[../../../microfluidics|microfluidics]]
